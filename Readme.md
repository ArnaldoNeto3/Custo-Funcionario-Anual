Custo do Funcionário anual

Este projeto lê um arquivo CSV contendo informações de funcionários, como nome, mês, horas trabalhadas e salário. A partir desses dados, o código calcula:

-> As horas trabalhadas por mês para cada funcionário

-> As horas totais trabalhadas no ano

-> O valor da hora (com base em 220 horas mensais)

-> O custo anual de cada funcionário considerando as horas trabalhadas

No final, o programa exibe o custo anual de cada colaborador já formatado.

Tecnologias utilizadas:

Python

Pandas para leitura e manipulação dos dados em CSV

Para executar o script, basta garantir que o arquivo base de dados.csv esteja no mesmo diretório e rodar o código em Python.
